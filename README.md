# weather-app

A web app developed using node.js. Use the search form to get the weather forecast of your location :)

## Instructions

1. Clone the repository locally.
2. From the terminal cd into the root folder and then run `cp .env.example .env`.
3. Set the environment variables.
4. Run `docker-compose up` to launch the application.
5. Enjoy :)

Based on the (The Complete Node.js Developer Course)[https://www.udemy.com/course/the-complete-nodejs-developer-course-2] course from Udemy, but with some small changes.
